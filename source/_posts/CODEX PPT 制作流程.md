## 第 1 步：先准备材料

准备以下内容：

- 论文 / 开题报告 / 实验数据（核心文本），发送给 GPT
    
- 答辩 PPT 页数要求（一般 12–16 页）
    
- 一张喜欢的学术风 PPT 参考图可在小红书 / 抖音搜索“毕业答辩 PPT”，保存一张作为风格参考。

---

## 第 2 步：用 GPT 文本模型生成答辩 PPT 大纲

打开 GPT，复制以下提示词：

```
我要做毕业答辩 PPT，基于我提供的论文材料生成 PPT 大纲。  
​  
要求：  
1. 严格按毕业论文答辩结构制作  
2. 总页数控制 xx 页以内  
3. 给出每一页精确标题  
4. 逻辑：封面 → 目录 → 研究背景 → 研究内容 → 研究方法 → 实验过程 → 实验结果 → 分析讨论 → 总结与展望 → 致谢  
5. 不编造内容，不脱离材料
```

把论文内容粘贴进去后，GPT 会输出完整大纲和每页标题。

你只需要核对：

- 页数是否合适
    
- 顺序是否合理
    
- 重点是否遗漏

---

## 第 3 步：让 GPT 生成每页具体内容

根据已经确定的大纲，继续让 GPT 生成每一页 PPT 的具体内容。

可复制以下提示词：

```
现在根据上面确定的大纲，生成每一页 PPT 的具体内容。  
​  
要求：  
1. 每页只写要点，不写大段文字  
2. 给出核心观点、3–5 条要点、建议插图  
3. 学术严谨，符合毕业答辩风格  
4. 输出格式：  
​  
第 x 页：标题  
核心观点：XXX  
内容要点：XXX  
建议插图：XXX
```

---

## 第 4 步：让 GPT img2 生成 PPT封面图片

必须先设置 2 个固定规则：

- 比例：16:9
    
- 语言：强制中文无乱码、黑体

直接复制下面这条 img2 万能提示词：

```
我要制作毕业答辩 PPT 封面，16:9 比例。  
请参考上传的图片风格，保持学术、简洁、正式。  
​  
当前页标题：【把你的封面标题粘贴进来】  
页面内容：【姓名、指导老师、学院、日期】  
​  
要求：  
1. 中文字体统一黑体、无乱码、无扭曲、无错字  
2. 排版整洁、留白充足  
3. 风格正式，适合毕业论文答辩  
4. 只做设计，不修改文字内容

生成后检查：

- 文字是否正确
    
- 有没有乱码
    
- 排版是否舒服
```

封面满意后，整套 PPT 的风格基本就确定了。

---

## 第 5 步：让 Codex 生成最终 PPT

将毕业论文 word 文件、第 3 步生成的每页具体内容、第 4 步生成的封面图片发给 Codex。

让 Codex 用 nature skill 生成可编辑的 ppt。

---

## （第 5 步：让 GPT img2 逐页生成所有 PPT 页面图片）

用第一页封面作为风格模板，逐页生成后续页面。

可复制以下提示词：

```
请完全延续封面页 PPT 的风格、配色、字体、版式质感，按照ppt大纲生成所有页面的图片，插入的图从毕业论文word中提取。  
​  
（复制PPT大纲）
​  
要求：  
1. 中文黑体、无乱码  
2. 图文并茂  
3. 学术清晰、矢量图质感  
4. 不编造内容、不改变标题

每生成一页都检查一遍：

- 文字是否正确
    
- 有没有乱码
    
- 插图是否贴合内容
    
- 页面是否拥挤
```

---

## （第 6 步：让 Codex 生成最终 PPT）

发送所有 PPT 页面图片和论文给 Codex，用 skill 转成可编辑的 PPT 文件。

---

## 只需要记住 3 条铁律

1. 先大纲 → 再内容 → 最后生图，不要一键生成
    
2. 每页必须检查中文，出现乱码立刻重新生成
    
3. 实验图绝对不要让 AI 重画
