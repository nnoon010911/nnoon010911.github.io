目前已经配置的 **MCP** 有：

1. **chrome-devtools-mcp**  
    用于连接 Chrome DevTools，实现网页调试、页面分析、自动化浏览等能力。  
    仓库地址： `https://github.com/ChromeDevTools/chrome-devtools-mcp`
2. **filesystem MCP**  
    用于访问和操作本地文件系统。  
    仓库地址： `https://github.com/modelcontextprotocol/servers`
3. **git / github MCP**  
    用于与 Git 和 GitHub 仓库交互，例如查看仓库内容、提交记录、代码管理等。  
    仓库地址： `https://github.com/modelcontextprotocol/servers`
4. **Zotero MCP**  
    用于连接 Zotero，支持文献管理、条目检索、参考文献辅助等功能。  
    仓库地址： `https://github.com/cookjohn/zotero-mcp`
5. **paper-search-mcp**  
    用于检索学术论文和相关科研资料。  
    仓库地址： `https://github.com/openags/paper-search-mcp`
6. **Draw. io MCP**  
    用于与 Draw. io 相关能力进行集成，支持图表绘制与图形化工作流。  
    仓库地址： `https://github.com/DayuanJiang/next-ai-draw-io`
